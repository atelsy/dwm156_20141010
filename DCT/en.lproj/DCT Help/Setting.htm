Setting

The following system settings can be changed in the setting dialog:

  • General: User can decide whether lock SIM card and change PIN and PIN2 or not.

  • Dialup: User can manage dialup service profiles.

    In this page, user can create a new dial-up profile for their mobile network operator, or modify, delete the existing profiles. Several default profiles are provided in Data Card Tool. If no suitable profile exists for the camped network, the user can add a new one to the connection list.

    To modify an existing profile, select its name and click the edit button. Change the necessary information and click the save button to replace the selected operator.

    To create a new network profile, click new button, enter the details and click save button then it'll be saved in profile list.

    To delete an existing profile, select its name and click delete button.

  • Network: The user can select network RAT mode and get current network information.

  • SMS: The user can select show notification when new message arrive or not, and play sound. Besides these, user can also specified sound to play.
