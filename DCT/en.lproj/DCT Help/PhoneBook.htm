PhoneBook

The Phonebook functions provide the capability to manage contact information on the Data Card and in a PC folder.  The user can retrieve the phonebook from the Data Card and back it up on the PC.  The user can also add, delete, and cut or copy contacts from one folder to another.

Phonebook Viewer

In the Phonebook Viewer, contacts are organized into different folders; the user can examine the contents of each contact using the intuitive graphical interface.  The user can also click on each column header to sort the contacts in ascending or descending order.

The Phonebook Viewer toolbar provides the following one-click functions:

  • Add a new contact to the PC or SIM Card.

  • Cut the selected contacts.

  • Copy the selected contacts.

  • Paste the cut or copied contacts to the specified folder.

  • Delete the selected contacts from the PC or SIM Card.

  • Download the contacts from the Data Card to PC side.  Following a successful download, the contacts are will in the SIM Card.
