Welcome to Data Card Tool

The Data Card Tool provides a variety of functions for a user to manage Data Card easily using a PC.  This application provides Toolbar options for the user on the top and displays the corresponding view on the below form view, once a selection has been selected.  Communication between the Data Card and the PC is achieved via a serial connection.

Data Card Tool provides the following function categories:

  • Phonebook is used to manage the contacts information on the Data Card and PC folder.  In addition to adding, editing and deleting contacts, Data Card Tool also allows the user to cut,copy and paste contact list information either in the PC folder or Data Card.

  • Message allows the user to read, edit and send SMS/EMS messages directly from the PC.

  • Dial up provides an easy way to browse web through Data Card.  It also supports creating profile for current network operator and dialup automatically.

  • Statistics It shows transmitted bytes, received bytes, elapse time and history flux statistics, and draws real-time data flow gram.

  • Settings contain General, Dialup, Network and SMS settings.
