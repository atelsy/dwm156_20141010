Select Contact(s)

When user click "TO" button in SMS Editor window, it will open "Select Contact(s)" window, and enables the user to select contact using an intuitive graphical interface. 

In the Select Contact(s) window, user can select contact list from "SIM Card", "PC Folder" or "Trash Can" by click the left panel, and it will show the contact list information in the right up list view, then user can either double click or select contact and then click "Add" button to add contact in the right bottom list. When all this is done, user can select click "OK" button or "Cancel" button to add or cancel the operation.
