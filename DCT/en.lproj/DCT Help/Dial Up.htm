Dial Up

The dial up enable the user to browse web through Data Card.  The user can create a new Connection or modify, delete an existing one.  Additionally, transmission of data between the PC and the Data Card is also supported.

Dial up Viewer

In the Dial up View, the user can choose approporate operator by name listed in the combobox and connect their computer to the Internet via the Data Card.

After PIN ready, if the option of automatically dial up is selected, then Data Card Tool will create a profile with current operator name and dial up automatically.
