SMS Editor

Message Editor enables the user to compose SMS messages using an intuitive graphical interface. 

In the message edit, user can compose message content as they wish. User can also add contact through click "TO" button to open contact select window. And after finish, user can click "Save" toolbar button to save the message either in PC folder or SIM Card.

The Message Editor Toolbar provides the following one-click functions:

  • Send the message.  A dialog box showing the sending progress and status appears.

  • Save the message.  A dialog box appears for the user to select the destination folder and memory type.

  • Cut the selected text in the recipient or message column.

  • Copy the selected text in the recipient or message column.

  • Paste the cut or copied text into the specified location.
