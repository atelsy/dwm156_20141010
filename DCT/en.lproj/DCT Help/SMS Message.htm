SMS Message

The Message functions access the SMS functions on the Data Card.  The user can compose a new message, send and receive text messages on the Data Card Tool.  Reply and forward functions for received messages are also supported.

Message Viewer

In the Message Viewer, messages are organized by memory type (Data card, PC); the user can examine each message in a familiar Windows Explorer style interface.  The user can also click on each column header to sort messages in ascending or descending order.

The Message Viewer toolbar provides the following one-click functions:

  • Compose a new message in the PC or SIM Card.

  • Reply to the selected received message.

  • Forward the selected received message.

  • Cut the selected messages.

  • Copy the selected messages.

  • Paste the cut or copied messages into a specific folder.

  • Delete the selected messages from the Handset and PC folders.

Download the messages from the Data Card to the PC.  After a successful download, the messages will be sorted according to their status and stored in the Handset\Inbox and Handset\Outbox folders.
