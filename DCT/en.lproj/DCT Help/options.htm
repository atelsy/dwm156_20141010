Options

User can set up Data Card Tool option through View/Option menu.

If "Disconnect dial-up connection on exit Data Card Tool" be selected, when exit Data Card Tool, it will auto disconnect dialup connection, otherwise, it will prompt user that it can not exit Data Card Tool.

If "Minimize to the tray on close Data Card Tool" be selected, when closing Data Card Tool, Data Card Tool will minimize to tray, otherwise, exit Data Card Tool.

If "Auto dialup after Data Card Tool start up" be selected, Data Card Tool will create a profile for current network operator and dialup automatically after camped on network.
